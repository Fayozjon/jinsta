# youneedfriends ?

## getting started

- install node
- clone this repo
- cd into folder
- `npm install`

create a file named `.env` with the following content:
```env
IG_USERNAME=insert_username_here
IG_PASSWORD=insert_password here
```

- `npm run dev` to start the bot
- adjust values in `src/constants.ts` to play around
